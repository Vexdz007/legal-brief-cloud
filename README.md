# Legal Brief in Cloud

Generate interactive legal briefs and create websites.

## Features

- Interactive legal brief generation
- Website creation capabilities
- Cloud-native architecture

## Quick Start

```bash
# Clone the repository
git clone https://github.com/Vexdz007/legal-brief-cloud.git
cd legal-brief-cloud

# Install dependencies
npm install

# Start development server
npm run dev
```

## Development

```bash
# Run tests
npm test

# Run linting
npm run lint

# Build for production
npm run build
```

## Deployment

This project uses GitHub Actions for CI/CD and deploys to Kubernetes.

See [.github/workflows/ci-cd.yml](.github/workflows/ci-cd.yml) for the pipeline configuration.
